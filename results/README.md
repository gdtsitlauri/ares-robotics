# Results Summary

This directory stores the committed ARES benchmark artifacts generated by the
repository's bounded experiment pipeline.

Current headline numbers:

- Control:
  `results/control/stability_analysis.csv` shows negative sampled Lyapunov
  decay for both LQR and the H-infinity-style surrogate controller.
- Planning:
  `results/planning/algorithm_comparison.csv` includes Dijkstra, A*, D* Lite,
  RRT, RRT*, and PRM across `50x50`, `100x100`, and `500x500` maps.
- Assembly:
  `results/assembly/latency_comparison.csv` reports `180 ns` for the assembly
  PID path versus `520 ns` for C and `3800 ns` for Python.
- Go:
  `results/go/fleet_benchmark.csv` scales from `1` to `50` robots with
  communication overhead rising from `1.1 ms` to `27.5 ms`.
- ARES-NAVIGATOR:
  `results/ares_navigator/navigation_success_rate.csv` reports `0.93` success
  rate for the hybrid policy versus `0.86` for MPC and `0.81` for RL alone.
- Simulation:
  `results/simulation/scenario_results.csv` contains scenario-level outcomes for
  maze navigation, dynamic obstacles, multi-robot coordination, and narrow
  passage driving.
